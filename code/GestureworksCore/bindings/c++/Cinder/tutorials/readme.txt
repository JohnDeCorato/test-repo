These tutorials have been written for the C++ Cinder framework and GestureWorks Core. 

The easiest way to use them:

1. Download Cinder:  http://libcinder.org/

2. Copy the GestureWorks DLL and GML file to c:\GestureWorks

3. Copy the contents of this folder into the Cinder\apps directory.

4. Open the sln file in the vc10 subfolder of each tutorial.
