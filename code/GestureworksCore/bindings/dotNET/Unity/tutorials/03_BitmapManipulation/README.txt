3 Interactive Bitmaps Tutorial

Please visit the following link to get started with this tutorial:  http://wiki.gestureworks.com/index.php/.NET_%26_Unity:_Interactive_Bitmaps

In order to run the finished example, you will need to copy the following files to the Assets/Plugins/Gestureworks/Core directory: 

GestureworksCore\bindings\dotNET\gwc_net\bin\x86\Release\GestureWorksCoreNET32.dll
GestureworksCore\GestureworksCore32.dll

And lastly, you will need to update the absolute paths for the core dll and gml in the Main.cs file on line 53 and 54.
