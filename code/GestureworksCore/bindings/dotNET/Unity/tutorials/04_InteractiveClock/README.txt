4 Interactive Clock Tutorial

Please visit the following link to get started with this tutorial:  http://wiki.gestureworks.com/index.php/.NET_%26_Unity:_Interactive_Clock

In order to run the finished example, you will need to copy the following files to the Assets/Plugins/Gestureworks/Core directory: 

GestureworksCore\bindings\dotNET\gwc_net\bin\x86\Release\GestureWorksCoreNET32.dll
GestureworksCore\GestureworksCore32.dll

NOTE: This tutorial builds with relative linking to the dll and gml resource unlike the previous ones.
